##### Criar arquivo texto:
```
cat > teste.txt
```

##### Conteudo de um arquivo:
```
cat teste.txt
```

##### Copiar conteúdo de um arquivo para outro:
```
cat teste.txt > teste02.txt
```

##### Copiar arquivo para outro Diretório:
```
cp teste.txt <path>
```

##### Mover arquivo para outro Diretório:
```
mv teste.txt <path>
```

##### Renomear arquivo:
```
mv teste.txt teste_b.txt
```
