

##### Criar arquivo texto:
```
cat > arq.txt
```

##### Conteudo de um arquivo:
```
cat Startup.cs
```

##### Copiar conteúdo de um arquivo para outro:
```
cat Startup.cs > NovoArquivo.cs
```

##### Mover arquivo para outro Diretório:
```
mv Program.cs Views/Home/
```

##### Renomear arquivo:
```
mv AppSettings.cs Settings.cs
```
